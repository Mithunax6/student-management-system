# student-management-system
This is a simple Student Management System project developed using Java.
The main idea of this project is to help manage student details in an easy way.
With this system, you can add new students, view their details, update information, and delete records whenever needed. 
It is a basic project created to understand how Java and Object-Oriented Programming (OOP) concepts work in real applications.

What this project can do:
1.Add new student details
2.View all students
3.Delete student records
4.Simple and easy-to-use menu system

Technologies Used:
1.Java
2.OOP (Object-Oriented Programming)

How to Run:
1.Download or clone this repository
2.Open the project in any Java IDE (Eclipse / IntelliJ)
3.Run the Main.java file
4.Follow the menu instructions
